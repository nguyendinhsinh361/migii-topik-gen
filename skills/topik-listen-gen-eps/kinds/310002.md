# KIND 310002 — Nhìn hình chọn đáp án (EPS, Level 3) [ẢNH]

Bao gồm variant: 310002, 3410003 (cùng cấu trúc, khác format đáp án).

## Thông tin cơ bản

| Trường | Giá trị |
|--------|---------|
| Title | `[5-9] 다음 그림을 보고 알맞은 대답을 고르십시오.` (310002) / `다음을 듣고 그림에 알맞은 답을 고르십시오.` (3410003) |
| Level | 3 |
| Độ khó | 2 — `image_qa` |
| Ngữ pháp | `informal_polite`, `formal_polite` |
| q_point | không |
| count_question | 1 |
| Loại ảnh | `object_scene_illustration` — 1 bức tranh minh họa cảnh |
| Số câu trong dữ liệu | 500 (310002=200, 3410003=300) |

## Chủ đề thường gặp

transport(5%), hobby_leisure(3%), weather_season(2%).

## Nhãn hệ thống

| Nhãn | Giá trị |
|------|---------|
| question_feature | `qf_image_qa` |
| audio_format | `audio_numbered` |
| answer_grammar | `ans_formal_polite` |

## Chiến lược bẫy

Không phát hiện bẫy rõ — đáp án nằm trong audio, không phải text options.

## Cấu trúc audio

- 4 câu mô tả được đọc to (đánh số 1-4), mô tả điều có thể đang xảy ra trong hình
- Format:
  ```
  1. 식사를 하고 있습니다.
  2. 빨래를 하고 있습니다.
  3. 머리를 자르고 있습니다.
  4. 김치를 만들고 있습니다.
  ```

## Cấu trúc đáp án

- q_answer: `["1. ", "2. ", "3. ", "4. "]` (310002) hoặc `["1.", "2.", "3.", "4."]` (3410003) — đáp án nằm trong audio
- q_text: có câu hỏi về hình (e.g., "사람들이 무엇을 하고 있습니까?")
- q_image: có (1 ảnh mỗi câu)

### Phân loại q_text (8 dạng)

| Dạng | Số câu | Ví dụ |
|------|:------:|-------|
| `object_what` | 94 | 이것은 무엇입니까? |
| `action_doing` | 111 | 이 사람은 무엇을 하고 있습니까? |
| `place_where` | 100 | 여기는 어디입니까? |
| `date_when` | 68 | 언제입니까? |
| `object_job` | 25 | 이 사람의 직업은 무엇입니까? |
| `weather` | 20 | 날씨가 어떻습니까? |
| `count` | 16 | 몇 개입니까? |
| `quality` | 23 | 어떤 것입니까? |

## Cách mô tả ảnh (q_image_description)

Mỗi câu có 1 bức tranh. Dùng key `"image"` để mô tả cảnh trong hình.

## Quy tắc tạo audio và hình

1. Hình phải mô tả cảnh rõ ràng, dễ nhận biết
2. Audio gồm 1 câu đúng + 3 câu hợp lý nhưng sai so với hình
3. 3 câu sai phải liên quan đến cùng chủ đề nhưng khác hành động/đối tượng/địa điểm

## Ví dụ

```json
{
  "g_text_audio": "1. 식사를 하고 있습니다.\n2. 빨래를 하고 있습니다.\n3. 머리를 자르고 있습니다.\n4. 김치를 만들고 있습니다.",
  "content": [{
    "q_text": "사람들이 무엇을 하고 있습니까?",
    "q_image_description": {
      "image": "여자가 부엌에서 빨래를 하고 있다. 세탁기 옆에 빨래 바구니가 있다."
    },
    "q_answer": ["1. ", "2. ", "3. ", "4. "],
    "q_correct": 2
  }]
}
```
